# ROOTS Villas — site + CMS

Site one-page pentru Roots Villas Brașov, cu panou de Admin integrat (conținut + SEO).

## Rulare locală
```bash
npm install
npm run dev
```
Apoi deschide http://localhost:5173. Butonul „Admin" e în stânga-jos.

Contextul complet al proiectului pentru Claude Code este în `CLAUDE.md`.
